TITLE: Denoise sensor signal (rolling median + moving average) (CSV -> CSV)
GOAL:
  Read CSV input_path with columns: timestamp (ISO-8601 string), value (float).
  Perform denoising in two steps:
    1) Rolling median filter with window_size samples.
    2) Moving average filter (centered) with window_size samples applied to the median-filtered signal.
  Write CSV output_path with columns: timestamp, value_raw, value_median, value_denoised.
  Return a summary dict with keys: n, window_size, rms_raw, rms_denoised.

ENTRY POINT (Python):
  def run(input_path: str, output_path: str, window_size: int = 5, *, seed: int | None = None) -> dict | None

INPUTS:
  - input_path:
      format: CSV (UTF-8, comma-separated)
      schema:
        - timestamp: str (ISO-8601, increasing)
        - value: float
      constraints:
        - at least window_size rows
        - values finite
  - window_size: int >= 3 and odd (if even, increase by 1)

OUTPUTS:
  - output_path:
      format: CSV
      schema:
        - timestamp: str
        - value_raw: float
        - value_median: float
        - value_denoised: float

NUMERICAL REQUIREMENTS:
  - Deterministic: yes.
  - No NaNs/Infs in outputs.

OPTIONAL PARTIAL CONSTRAINTS / PROPERTIES:
  - rms_denoised <= rms_raw (within rel_tol=1e-12)

ALLOWED LIBRARIES:
  numpy, pandas

@params: window_size=5
@output_schema: timestamp,value_raw,value_median,value_denoised

@check: finite(value_raw,value_denoised)
@check: rms(value_denoised) <= rms(value_raw) rel_tol=1e-12
