TITLE: Resample irregular time series to regular grid (CSV -> CSV)
GOAL:
  Read CSV input_path with columns: t (float seconds), v (float).
  The time stamps are irregular and increasing.
  Create a regular time grid from min(t) to max(t) with step dt.
  Interpolate v onto this grid using linear interpolation.
  Write CSV output_path with columns: t, v_interp.
  Return a summary dict with keys: n_in, n_out, dt.

ENTRY POINT (Python):
  def run(input_path: str, output_path: str, dt: float = 0.5, *, seed: int | None = None) -> dict | None

INPUTS:
  - input_path:
      format: CSV
      schema: t: float, v: float
      constraints: t strictly increasing, dt > 0

OUTPUTS:
  - output_path:
      format: CSV
      schema: t: float, v_interp: float

NUMERICAL REQUIREMENTS:
  - Deterministic: yes.
  - No NaNs in v_interp.

OPTIONAL PARTIAL CONSTRAINTS / PROPERTIES:
  - v_interp at original t values equals original v within abs_tol=1e-12 (when grid includes those t values).

ALLOWED LIBRARIES:
  numpy, pandas, scipy

INPUT FILES PROVIDED WITH THIS EXAMPLE:
  - input_03_irregular.csv
